Een web-applicatie die leerlingen helpt veilig van school naar huis te komen.

## Gebruik

1. Open `index.html` in een moderne webbrowser
2. Geef toestemming voor locatie-toegang wanneer daarom wordt gevraagd
3. Stel je thuislocatie in via "Thuislocatie instellen"
4. Gebruik "Route naar huis" om een route te berekenen
5. Meld gevaarlijke plekken om anderen te helpen

## Technologie

- HTML5
- CSS3
- JavaScript (Vanilla)
- Leaflet.js (voor kaarten)
- Leaflet Routing Machine (voor routes)
- OpenStreetMap (kaartdata)
- LocalStorage (voor data opslag)

## Opmerkingen

- De app werkt het beste in Chrome, Firefox of Edge
- GPS locatie vereist toestemming van de gebruiker
- Routes worden berekend via OpenStreetMap routing service
- Gegevens worden lokaal opgeslagen in de browser
